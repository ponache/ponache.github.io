# Portfolio — Николай Щекочихин

Персональный сайт-портфолио продуктового дизайнера. **Читай этот файл первым** — здесь всё, что нужно, чтобы не сканировать проект заново.

## Что это
Статический сайт: чистый HTML + CSS + ванильный JS. **Нет** сборки, фреймворков и зависимостей. Правишь файл → обновляешь браузер. Превью: `open index.html`.

## Файлы
| Файл | Что внутри |
|------|-----------|
| `index.html` | Главная / «Работы»: hero, лента проектов, info-строки, футер |
| `about.html` | «Обо мне»: био, портрет, info-строки, футер |
| `projects/project-template.html` | Шаблон страницы кейса (копируется под каждый проект) |
| `styles.css` | Все стили. Токены в `:root`, секции размечены большими комментами-заголовками |
| `main.js` | Все интеракции (одна IIFE): переключение языка, scroll-reveal, липкий хедер, бургер-меню, модалка наград, год в футере |
| `CONTENT.md` | Карта всего текста/контента — для отдельного диалога по копирайту (см. ниже) |

## Правила
- Держим проект **без зависимостей и без сборки**. Не тащим npm/фреймворки.
- Стиль кода: отступ 2 пробела; комментарии часто на русском — сохраняй тон.
- Все стили — в `styles.css` через CSS-переменные из `:root` (акцент `--brand: #656AED`, скругления, easing `--t-fast`/`--t-slow`). Не хардкодь значения, если есть токен.
- Двуязычность RU/EN: у каждого переводимого узла атрибуты `data-ru` / `data-en`; атрибуты (href и т.п.) локализуются через `data-ru-<attr>` / `data-en-<attr>`. Язык по умолчанию — EN, хранится в localStorage `portfolio-lang`.
- Шрифты: системный SF Pro → Inter (Google Fonts) как фолбэк.
- Ключевые брейкпоинты: **760px** (хедер/мобилка), **720px** (info-строки), **640px** (модалка → нижний шит), **600px** (лента → 1 колонка).
- Стейты нажатия кастомные (`:active`: scale + лёгкое затемнение текста). Дефолтная мобильная подсветка тапа отключена глобально (`-webkit-tap-highlight-color: transparent`) — не возвращай её.

## Модалка наград (index + about)
- Кнопки `[data-modal-open="aw-1|aw-2|aw-3"]` открывают `.modal-overlay#aw-x`.
- **Десктоп**: карточка справа-сверху, появляется плавно **без движения** (только opacity), клик по любому месту закрывает.
- **Мобилка (≤640px)**: нижний шит, свайп вниз **за любую область** шита закрывает (логика в `initModal` → `touchstart/move/end`).
